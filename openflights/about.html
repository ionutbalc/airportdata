About

OpenFlights is a tool that lets you map your flights around the world, search and filter them in all sorts of interesting ways, and calculate statistics automatically. It's also the name of the open-source project to build the tool. Read more about it in the FAQ.

OpenFlights is presently a gamma release — that means the site is now open to the public and the basic features are all there, but work is still in progress on adding new features. If you'd like to be informed about the official launch, drop us a line.
