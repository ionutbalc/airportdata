Context Help: Comma-Separated Value (CSV) Format

This document defines Revision 1.0 of the comma-separated value (CSV) format natively exported and imported by OpenFlights.

OpenFlights CSV follows RFC 4180.

  • The field separator is comma (, 0x2C), the row separator is CRLF (\r\n 0x0D 0x0A) and the field delimiter is double quote (" 0x22).
  • The first line of any OpenFlights CSV file is considered a header and ignored.
  • All fields may be left empty unless listed as Mandatory below.

Field definition

1. Date
Date of flight, in YYYY-MM-DD format. Mandatory.
2. From
IATA or ICAO code of source airport. Mandatory unless "From_Code" is provided.
3. To
IATA or ICAO code of destination airport. Mandatory unless "To_Code" is provided.
4. Flight_Number
Flight number. Mandatory if "Airline" or "Airline_Code" is not provided. Overrides "Airline" if first two characters are an airline code (two alphanumeric digits).
5. Airline
Airline name. Mandatory if "Flight Number" with airline code or "Airline_Code" is not provided. Looked up from database if "Flight Number" or "Airline_Code" provides airline code.
6. Distance
Distance of flight, in miles. Calculated automatically if not provided.
7. Duration
Duration of flight, in the format HH:MM. Estimated automatically if not provided.
8. Seat
Seat number.
9. Seat_Type
Seat type: one of "W" for Window, "A" for Aisle or "M" for Middle.
10. Class
Class: one of "F" for First, "B" for Business, "P" for Premium Economy, "Y" for Economy.
11. Reason
Reason for flight: one of "B" for Business, "L" for Personal, "C" for Crew or "O" for Other.
12. Plane
Plane type.
13. Registration
Plane registration number.
14. Trip
Internal OpenFlights Trip ID. Optional, but must exist and belong to the user importing the data if specified.
15. Note
User-entered note or comment about flight.
The final four fields contain internal OpenFlights data for ensuring accurate imports. They are generated automatically when exporting, but should not be provided if creating or importing your own data.
16. From_Code
Internal OpenFlights airport ID. Overrides "From" if provided.
17. To_Code
Internal OpenFlights airport ID. Overrides "To" if provided.
18. Airline_Code
Internal OpenFlights airline ID. Overrides "Airline" if provided.
19. Plane_Code
Internal OpenFlights plane ID. Overrides "Plane" if provided.

Sample

The following sample shows the maximal, recommended, and minimal data sets for importing a flight.
