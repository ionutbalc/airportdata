Context Help: Airline

"Airline" is the airline you used for this flight.


If you have already flown with this airline, you can choose it from the drop-down box.


If you need to enter a new airline, enter the airline name or a part of it (eg. "Cathay Pacific", "Cathay", "Cath") or the airline code (2-letter IATA, "CX", or 3-letter ICAO, "CPA") into the search box, and press TAB or ENTER. A list of matching airlines will appear in the drop down box.


Tip: If you enter a flight number like "CX123", OpenFlights will automatically match "CX" to Cathay Pacific and fill in the airline for you.


Tip: If you're searching for an obscure airline (with no IATA code), enter the ICAO code or at least four characters to trigger a full-database search. If you still can't find it, contact OpenFlights.
