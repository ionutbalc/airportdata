OpenFlights: Forgot password?

One tip before you panic: make sure you have entered your password using the same case as you did when signing up: "Secret" is not the same as "secret". If you signed up before January 15, 2009, your username ("Joe" vs "joe") also has to match.

If that doesn't help and you gave us your e-mail address when signing up, please send mail to support at openflights dot org, give your account name and request to have your password reset. A new password will be mailed to the address you gave us within a few days.

If you didn't register an e-mail address, then sorry, we don't know that you're you, so there's really nothing we can do.
