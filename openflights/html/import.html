OpenFlights: Import data

OpenFlights can import two types of data:

FlightMemory (.html) — FlightMemory does not have an export function, but OpenFlights can "screen-scrape" its FlightData pages. To create one, log into your FlightMemory account, go to the FlightData page, and select "Save Web Page" in your browser. Upload the resulting HTML file. Repeat once for each page of data.


Note: The airline names used on FlightMemory do not always match those used by OpenFlights, but the import tool will try its best to match them. Importing accuracy is much higher if your FlightData also includes flight numbers; for example, there are four airlines named "Virgin", but only Virgin Atlantic has the code "VS".

Comma-separated value (.csv) — Easily exported from Excel or any other spreadsheet. Coming soon...
